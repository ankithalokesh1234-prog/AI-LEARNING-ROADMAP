# Day 2 - Python Basics

## Variables
Variables are used to store data.

Example:
name = "Ankitha"

## Data Types

-->Integer (int)
      Whole numbers.
      Example:
      x = 10
-->Float
      Decimal numbers.
      Example:
      price = 99.99
--> String (str)
      Text data.
      Example:
      name = "Python"
--> Boolean (bool)
      True or False values.
      Example:
      is_student = True

## Input

Used to take input from the user.
name = input("Enter your name: ")

## Output

Used to display output.
print("Hello")

## Operators

-->Arithmetic Operators
      +  -  *  /  %  //  **

--> Comparison Operators
      ==  !=  >  <  >=  <=

--> Logical Operators
    and  or  not

## Type Conversion
age = int(input("Enter age: "))
Converts input from string to integer.
