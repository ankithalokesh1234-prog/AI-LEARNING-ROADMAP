# Day 5 - Lists, Tuples, Dictionaries and Sets

## Lists
A list is an ordered and mutable collection used to store multiple items. Lists allow duplicate values.

### Example
```python
fruits = ["apple", "banana", "mango"]

## Tuples
A tuple is an ordered and immutable collection. Once created, its values cannot be changed.

### Example
```python
colors = ("red", "green", "blue")
print(colors)
```

Output:
```text
('red', 'green', 'blue')
```

---

## Dictionaries
A dictionary stores data in key-value pairs. Each key must be unique.

### Example
```python
student = {
    "name": "Ankitha",
    "age": 19
}
print(student)
```

Output:
```text
{'name': 'Ankitha', 'age': 19}
```

---

## Sets
A set is an unordered collection of unique values. Duplicate values are automatically removed.

### Example
```python
numbers = {1, 2, 3, 2, 1}
print(numbers)
```

Output:
```text
{1, 2, 3}
```
